# Triage Labels

| Role | Label string |
|------|-------------|
| Maintainer needs to evaluate | `needs-triage` |
| Waiting on reporter | `needs-info` |
| Fully specified, ready for an AFK agent | `ready-for-agent` |
| Needs human implementation | `ready-for-human` |
| Will not be actioned | `wontfix` |

These labels should exist in the GitHub repo. Create them with:
`gh label create <name> --repo boytheerapol/JotHai`
