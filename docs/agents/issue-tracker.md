# Issue Tracker

**Tracker:** GitHub Issues
**Repo:** boytheerapol/JotHai
**CLI:** `gh` (GitHub CLI)

## Workflow

- New issues: `gh issue create --repo boytheerapol/JotHai`
- List issues: `gh issue list --repo boytheerapol/JotHai`
- Update labels: `gh issue edit <number> --add-label <label>`
- Close: `gh issue close <number>`

## PR triage surface

External PRs are **not** a triage surface. Do not pull PRs into the triage queue. Collaborators' in-flight PRs are left alone.
